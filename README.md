# chatgpt-review-analysis
Sentiment analysis of ChatGPT user reviews
## Project Objective
Analyze ChatGPT user reviews to understand:
- Sentiment distribution (Positive/Neutral/Negative)
- Key issues in negative reviews
- Positive feature keywords
- Sentiment trends over time

## Dataset Columns
- Review ID
- Review
- Ratings (0â€“5)
- Review Date

## Techniques Used
- Data cleaning & preprocessing
- Sentiment analysis using TextBlob (polarity, subjectivity)
- Keyword extraction (positive & negative)
- Visualizations (bar charts, wordcloud, time series)

## Output
- Sentiment distribution chart
- Ratings vs sentiment chart
- Negative issue keywords chart
- Positive wordcloud
- Monthly sentiment trend

## Tools
- Python
- Google Colab
- Pandas, Matplotlib
- TextBlob, WordCloud
